# Self-hosted Google Fonts

This directory caches the font CSS and `woff2` files used by the blog design.

Source families:

- Literata
- Source Sans 3
- Noto Serif SC
- Noto Sans SC

The files were downloaded from Google Fonts and are referenced by `fonts.css` with relative URLs so the site works under the GitHub Pages project path.

Licenses:

- Literata, Noto Serif SC, and Noto Sans SC are distributed under the SIL Open Font License.
- Source Sans 3 is distributed under the SIL Open Font License.

One unused Literata Greek subset was omitted after a transient network download failure. The site is Chinese-first and falls back to system fonts for that range if needed.
